dmbm: Browser-agnostic bookmarking

(Repository: https://codeberg.org/cyberme0w/dmbm)

dmbm is a bookmarking extension for dmenu, a launcher for tiling window managers such as i3 or dwm.

It is not bound to any browser and uses xdotool, meaning you can use it across multiple browsers, as well as any other program that accepts cursor input. It also works well enough as a copy-pasta board.

It is written in bash, which makes it easy to extend, and stores bookmarks in plain text files within the bookmarks‚Äô folder-hierarchy.
